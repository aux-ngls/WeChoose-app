# Reliure Mobile

Application Expo/React Native de Reliure, derivee de la structure mobile Qulte.

## Lancer l'app

```bash
cd ~/Qulte/reliure/mobile
npm start
```

La commande lance Expo comme Qulte. Reliure mobile utilise par defaut l'API en ligne:

```bash
https://api.wechoose.dury.dev/reliure
```

Puis:

- scanner le QR code avec Expo Go
- ou appuyer sur `a` pour Android
- ou appuyer sur `i` pour iPhone si le simulateur iOS est disponible

## Mises a jour

```bash
cd ~/Qulte/reliure/mobile
npm run update
```

## Backend local optionnel

```bash
cd ~/Qulte/reliure/mobile
npm run dev:local
```
