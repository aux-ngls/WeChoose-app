# Qulte - Roadmap

Last updated: 2026-05-04

## Maintenance Rule

This roadmap must be updated whenever priorities change, a major milestone is completed, or a new workstream becomes important.

It is part of the shared project memory and should stay aligned with `PROJECT_MEMORY.md`, `DECISIONS.md`, and `RELEASES.md` so every future conversation starts from the same context.

## Immediate Priorities

- Keep removing visible loading states, especially in Tinder and high-traffic mobile screens.
- Improve first connection experience and onboarding quality.
- Continue improving recommendation quality using onboarding signals, ratings, and behavior.
- Stabilize push notifications outside the app.
- Improve TestFlight update workflow and release confidence.

## Mobile Product Priorities

- Make all core flows feel native and fluid.
- Keep message sending instant and optimistic.
- Keep movie details consistent everywhere.
- Improve profile discovery and user search.
- Improve review editing, deletion, comments, and profile display.
- Make playlist management fast and clear.

## Recommendation Priorities

- Better separate Tinder recommendations from other recommendation surfaces.
- Avoid proposing onboarding-selected movies back to the user.
- Increase quality weighting through public ratings without making results too mainstream.
- Add controlled exploration to avoid repetitive recommendations.
- Measure recommendation outcomes per user where possible.

## Social Priorities

- Better user discovery.
- Better profile pages.
- Better review conversations.
- Sharing movies should be quick from movie details and messages.
- Notifications should reliably work for private messages.

## Commercial Readiness Priorities

- Reliability and crash reduction.
- Clear onboarding.
- Strong recommendation quality from first session.
- App Store / TestFlight polish.
- Privacy and data ownership clarity.
- Basic moderation/reporting for social content before wider launch.

## Later Ideas

- Group movie matching.
- Shared watchlists.
- Better friend-based discovery, without overpowering personal taste.
- Public profile links.
- More advanced recommendation explanations.
- Analytics dashboard for recommendation quality.
